## Description

- **Phase 1 Interactions:**
  - Draw Area: Click deck or discard pile.
  - Card Holding: Show the two drawn cards.
  - Play/Discard: Drag card to a stack or discard area.
- **State Sync:** Reflect the game state from the backend WebSocket in real-time.

## Definition of Done
- Fully playable Phase 1 in the browser.
- Smooth animations for drawing and playing cards.
