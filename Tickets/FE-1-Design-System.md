# FE-1: Design System

## Goal
Establish the project's visual identity and core components.

## Prerequisites
None.

## Description
- Initialize Vite + React + Vanilla CSS in `shovels_frontend/`.
- Create `index.css` with a "Premium/Dark/Tactile" design system.
- **Tokens (Use these specifically):**
  - **Background**: `#0F0F0F` (Dark Matte)
  - **Surface**: `#1A1A1A` with subtle borders (`#333`)
  - **Primary (Gold)**: `#D4AF37` (Accent for Kings/Winners)
  - **Secondary (Crimson)**: `#8B0000` (Hearts/Damage)
  - **Typography**: `Outfit` or `Inter` from Google Fonts.
- **Core Components:**
  - `Card`: Visual representation using SVG for pips/faces.
  - `Stack`: A fanned-out vertical list of cards with hover magnification.
  - `Button`: Tactile feel (inset on click, glow on hover).

## Definition of Done
- A Style Guide page showing all base components.
- Responsive layout skeleton (Sidebar, Main Content).
