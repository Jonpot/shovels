# FE-4: Phase 2 UI

## Goal
Build the Battle interface with full interaction for stacks and powers.

## Prerequisites
- [FE-3: Phase 1 UI](file:///c:/Users/jonat/Desktop/Personal/shovels/Tickets/FE-3-Phase-1-UI.md)
- [CORE-3: Phase 2 Actions](file:///c:/Users/jonat/Desktop/Personal/shovels/Tickets/CORE-3-Phase-2-Actions.md)

## Description
- **Stack Visualization:** Vertical fan-out of cards. Numbers should be clearly visible.
- **Battle Interactions:**
  - Select character to act.
  - Multi-select cards from top of stack for "Action Hand".
  - Select target character for attacks.
  - Drag-and-drop support for Shop purchases.
- **Visual Feedback:**
  - Indicators for "Tapped" characters.
  - Damage animations (shaking, removal effects).
- **Shop Row:** Display the 3 shop cards and refresh button.

## Definition of Done
- Fully playable Phase 2 combat.
- Support for complex Spade/Shovel digging UI flows.
